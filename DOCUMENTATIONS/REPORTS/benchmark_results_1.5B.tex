\section*{Summary of Findings}
This document compares the locally replicated benchmarks against the original research paper results. 
\begin{itemize}
    \item \textbf{Baseline:} Local BF16 baseline outperformed the paper's baseline (+0.71 Avg).
    \item \textbf{Critical Failures:} All \textbf{QuaRot} variants (KV4, KV3, W8A8, W4A4) failed catastrophically (~0.6 Avg), consistent with paper's KV-Cache QuaRot but contradicting paper's Weight+Act QuaRot results.
    \item \textbf{Metric Adjustment:} The paper's "Avg" originally included GPQA. For this comparison, the paper's average was \underline{recalculated} using only the 4 common metrics (AIME, MATH, GSM8K, LCB).
\end{itemize}
